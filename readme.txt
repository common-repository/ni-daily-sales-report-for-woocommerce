Ni Daily Sales Report for WooCommerce offers comprehensive sales analysis for any sales period.

== Description ==

<p>The Daily Sales Report offers a comprehensive overview of sales activities, encompassing orders, ordered products, and refunds within the specified date range. Users can easily search the report by entering the start and end dates, accessing detailed sales summaries for the selected period.</p>


<h4>Key Features of Ni Daily Sales Report For WooCommerce</h4>
<ul>
<li>Filter By Date Range</li>
<li>Show Order Totals Including Order Total, Order Tax, Cart Discount, and Other Sales Totals</li>
<li>Order Status Total Sales</li>
<li>Product Line Total Sales</li>
<li>Refund Totals</li>
</ul>

== Installation ==

1. Use the WordPress plugin installer or upload files to WordPress plugin directory.
2. Activate "Ni Daily Sales Report For Woocommerce" plugin from admin panel.
3. Find admin menu "Daily Sales Report".
